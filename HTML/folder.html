Front-end web
development


Estructurar el diseño de la página web con código HTML, darle un estilo unico con CSS y agregarle dinamismo con ayuda de JavaScript, son habilidades que he aprendido a lo largo de el tiempo, tratando la menera de mejorar siempre la escritura del código de todo el contenido semánticamente para que la página cuente con una confiable y segura accesibilidad cuando se navegue en sus diferentes contenidos requeridos, optimizando los tiempos de transferencia de datos sin afectar la calidad de los mismos.

Adaptar el diseño web a diversos dispositivos y navegadores (Responsive Web Design) es una tarea siempre presente en un buen programador en estos tiempos, ya que todo mundo navega desde sus smartphone o tablets y tener esa sensación al ingresar a su navegador y encontrarse con páginas como que si fuesen otra aplicación más y no de una página web solo pensada para ordenadores de escritorio, es algo que muchos agradecen, ya que la barra de navegacion, banners, cards, footer se adaptan de una forma mágica para el usuario y sus diferentes tamaños de pantallas de dispositivos que existen.

Para el layout puedo trabajar con Flexbox y Grid.

Con este lenguaje puedo trabajar tanto en la parte visual como en lo que no se ve, pero que tambien esta alli para funcionar en nuestras aplicaciones, puedo desarrollar con la ayuda de el Storyboard para colocar iconos, tab-bar, botones, tablas, texto etc., igualmente desarrollar toda la aplicación unicamente utilizando código, o combinar ambas como en la aplicación de chat de las imagenes, la aplicación de la red social es completamente código.

Con esta aplicación de mensajeria logre establecer comunicación en tiempo real gracias a Firebase Database, y crear una aplicación de mensajes en la cual los usuarios pueden registrarse con un correo y contraseña o autenticarse con su cuenta de Facebook o Google, para poder utilizarla y enviar todo tipo de información (escrito, visual (fotos y videos) o audio) asi como también enviar a otro usuario su ubicación utilizando la Geolocalizacion, tarde aproximadamente tres meses en terminarla ya que estaba entendiendo bien la forma de estructurar la funciones y los protocolos en el backend para optimizar tanto la emisión como la recepción de la información.

También desarrolle otras aplicaciones con API Rest para saber el estado del clima y una de un juegos de trivias con banderas, otra que solicitaba peliculas en renta a una empres en España, he mejorado muchismo con el diseño en las aplicaciones al igual que con la estructuración de el código trabajando con las arquitecturas MVC y MVVM, realizando una escritura de código más entendible y ordenada.
